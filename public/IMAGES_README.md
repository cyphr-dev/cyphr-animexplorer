# Required Images for PWA and OpenGraph

Please add the following images to the `/public` directory:

## PWA Icons

- **icon-192.png** (192x192px) - App icon for mobile devices
- **icon-512.png** (512x512px) - Large app icon
- **apple-icon.png** (180x180px) - Apple Touch Icon
- **favicon.ico** - Website favicon

## OpenGraph

- **og-image.png** (1200x630px) - Social media preview image

## Optional Screenshots (for better PWA experience)

- **screenshot-desktop.png** (1280x720px) - Desktop app screenshot
- **screenshot-mobile.png** (750x1334px) - Mobile app screenshot

## Image Guidelines:

1. Icons should have a consistent design and work on both light and dark backgrounds
2. Use your app's branding colors and logo
3. OG image should be eye-catching and represent your app's purpose
4. Screenshots should show the actual app interface in use

## Tools to Create Icons:

- [Favicon Generator](https://realfavicongenerator.net/)
- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [Canva](https://www.canva.com/) for OG images
